## 前端题目

### Your Idol

制作一个网页，向大家介绍你的偶像，可以是明星、作家、音乐家、动漫人物、家人、甚至你自己......发挥想象，为“他”打call！

### 基础要求

1、页面整洁美观，代码条理清晰

2、包含图片、矢量图形、链接等因素

3、使用css样式修饰页面

4、无空链接、表格错位、乱码等错误

### 进阶要求

1、编写JavaScript 代码，实现一些高级功能

2、有独特的设计思路，富有有个人特色

3、内容原创，具有观赏性、可读性，主题突出

#### 注意事项：
我们统一使用谷歌浏览器作为网页浏览的标准。
