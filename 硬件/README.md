## 【硬件 · 基础题：键控闪烁灯】

---
### 【赛题说明】  
- 制作一个线性稳压电源，12V学生电源输入，给各模块电路供电，输出电压3V~5V可调，空载电压纹波峰峰值小于0.1V。  
- 制作一个单稳态多谐振荡器，由按键控制，脉宽1~3s可调。控制LED1的亮灭。  
- 在单稳态多谐振荡器的基础上制作一个多谐振荡器，震荡频率1~20Hz可调，占空比50%，误差加减10%。控制LED2的亮灭。

### 【注意事项】

- 设计报告采用打印形式。
