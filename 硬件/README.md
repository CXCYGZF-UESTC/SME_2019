## 硬件题目(2)

### 简易通断测试仪

#### 基础要求

1.	在±10V供电下制作可以驱动制作可以驱动100Ω负载的3mA电流源

2.	负载在50Ω~100Ω变化时电流误差不大于5%

3.	在±8V~±12V供电，100Ω负载下电流误差不大于3%

4.	在负载<60Ω时进行声光提示

#### 进阶要求

1.	更低的负载，更高的精度，更小的电流

2.	报告

3.	创新的设计
